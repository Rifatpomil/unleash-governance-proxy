## Steps

1. **Create a new repo on GitHub**
   - Go to https://github.com/new
   - Name: `unleash-governance-proxy` (or whatever you prefer)
   - Make it **empty** (no README, no .gitignore)
   - Create it

2. **Add remote and push**
   ```powershell
   cd e:\PersonalProjects\Rifat\unleash-governance-proxy-new

   git remote add origin https://github.com/Rifatpomil/unleash-governance-proxy.git
   git branch -M main
   git push -u origin main
   ```

   (Replace `Rifatpomil/unleash-governance-proxy` with your actual repo path)

3. GitHub will show **Rifat Alam Pomil** as the sole contributor. Done.
